【剑指offer】矩形覆盖

今天继续介绍一道递推的题目。

矩形覆盖

我们可以用2x1的小矩形横着或者竖着去覆盖更大的矩形。请问用n个2x1的小矩形无重叠地覆盖一个2xn的大矩形，总共有多少种方法？

思路

考虑最后一个竖着的2x1方块，它有2中可能：1、被一块竖着的2x1填满；2、被2个横着的1x2的后板块填满。其实对于第1种情况，共有F(n-1)中可能；对于第2种情况，共有F(n-2)中可能。所以的到递推公式 F(n) = F(n-2) + F(n-1)。
n = 1时，F(1) = 1;
n = 2时，F(2) = 2。
