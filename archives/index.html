在ubuntu上安装jdk


最近需要在阿里云的服务器上运行一些自己的小demo，选择了比较熟悉的ubuntu系统。为了能让这些小工程能顺利运行起来，需要先安装一些javaWeb工程运行的环境软件，今天先记录下怎么安装jdk，以备后期查询。

1、创建目录

2、上传文件

把jdk-7u75-linux-x64.tar.gz文件上传到/usr/lib/jvm目录（实现从oracle网站上下载好）。

3、转到目录

4、解压文件

5、重命名

6、环境变量

在最后增加如下内容：
export JAVA_HOME=/usr/lib/jvm/java
export JRE_HOME=${JAVA_HOME}/jre
export CLASSPATH=.:${JAVA_HOME}/lib:${JRE_HOME}/lib
export PATH=${JAVA_HOME}/bin:$PATH

7、配置默认JDK版本

如果只有1个 JDK可选，提示
There is only one alternative in link group java (providing /usr/bin/java): /usr/lib/jvm/java/bin/java
无需理会。

8、测试

如果能打印出jdk版本1.7.0_75，表示安装成功
