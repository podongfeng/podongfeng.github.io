【使用hexo搭建博客（一）】Hello Hexo

今天是羊年大年初一。
昨天还跟小伙伴说新年要开始写一点技术博客，今天下午利用难得的休闲时光，就在网上找了找博客的框架。本来是想用wordpress的，都看了一些教程了，偶然发现了这个东西——hexo，简单看了一下介绍，发现hexo更加符合我的需求。
下面是hexo官网介绍：
A fast, simple & powerful blog framework, powered by Node.js.
官网的页面和风格和这个介绍同样简洁，感觉也蛮符合现在的趋势。

Talk is cheap, 马上动手！

这里说一下上面2个命令其实是简写：
hexo g = hexo generate
hexo s = hexo server
从意思上也比较好理解，generate就是生成的意思，就是生成静态的博客页面；server就是提供服务，这样上一步生成的页面就可以被访问到了。
好了，今天就先写到这边吧，我们已经成功使用hexo在本机搭建了博客，后面再讲如果做一些个性化的配置，还有就是把这个部署到Github Page上面去，毕竟，博客是要被别人访问到的，只是自己在本机上面玩就没多大意思了。
